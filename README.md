a7f4-linux-configs
====================


Please edit the `rc.lua` file in order to fix the path to the theme! Look at variable named `themePath`. That usually should point to somewhere where you have your checkout. 

Requirements
------------

* awesome 3.5
* python-distutils-extra
* wmname
* synapse
* numlockx
* nemo
* volumeicon
* xxkb
* hamster-indicator
* copyq
